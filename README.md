# rudolf

Themed WhatsApp desktop client made with Electron.

![Main menu](/screenshots/screenshot2.png?raw=true "Contacts aren't blurred when hovering over them.")

# Build

```
npm i
sudo npm i -g electron-forge
electron-forge make
```
